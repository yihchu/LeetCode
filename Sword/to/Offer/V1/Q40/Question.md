<h1>最小的k个数</h1>

输入整数数组arr，找出其中最小的k个数。例如，输入4、5、1、6、2、7、3、8这8个数字，则最小的4个数字是1、2、3、4。</br>

示例1：</br>
输入：arr = [3, 2, 1], k = 2</br>
输出：[1, 2]或者[2, 1]</br>

示例2：</br>
输入：arr = [0, 1, 2, 1], k = 1</br>
输出：[0]</br>

限制：</br>
0 <= k <= arr.length <= 10000</br>
0 <= arr[i]<= 10000</br>
