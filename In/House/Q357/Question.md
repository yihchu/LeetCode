<h1>统计各位数字都不同的数字个数</h1>

给你一个整数n，统计并返回各位数字都不同的数字x的个数，其中0 <= x < 10^n。</br>

示例1：</br>
输入：n = 2</br>
输出：91</br>
解释：答案应为除去11、22、33、44、55、66、77、88、99外，在0 ≤ x < 100范围内的所有数字。</br>

示例2：</br>
输入：n = 0</br>
输出：1</br>

提示：
- 0 <= n <= 8
