<h1>在每个树行中找最大值</h1>

给定一棵二叉树的根节点root，请找出该二叉树中每一层的最大值。</br>

示例1：</br>
</br>![](./image/1.jpeg)</br></br>
输入: root = [1, 3, 2, 5, 3, null, 9]</br>
输出: [1, 3, 9]</br>

示例2：</br>
输入: root = [1, 2, 3]</br>
输出: [1, 3]</br>

提示：
- 二叉树的节点个数的范围是 [0, 10^4]
- -2^31 <= Node.val <= 2^31 - 1
