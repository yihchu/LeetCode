<h1>三数之和</h1>

给你一个包含n个整数的数组nums，判断nums中是否存在三个元素a，b，c，使得a + b + c = 0 ？请你找出所有和为 且不重复的三元组。</br>
注意：答案中不可以包含重复的三元组。</br>

示例1：</br>
输入：nums = [-1, 0, 1, 2, -1, -4]</br>
输出：[[-1, -1, 2], [-1, 0, 1]]</br>

示例2：</br>
输入：nums = []</br>
输出：[]</br>

示例3：</br>
输入：nums = [0]</br>
输出：[]</br>

提示：</br>
0 <= nums.length <= 3000</br>
-10^5 <= nums[i] <= 10^5</br>
