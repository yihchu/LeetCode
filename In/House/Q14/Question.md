<h1>最长公共前缀</h1>

编写一个函数来查找字符串数组中的最长公共前缀。</br>
如果不存在公共前缀，返回空字符串""。</br>

示例1：</br>
输入：strs = ["flower", "flow", "flight"]</br>
输出："fl"</br>

示例2：</br>
输入：strs = ["dog", "racecar", "car"]</br>
输出：""</br>
解释：输入不存在公共前缀。</br>

提示：</br>
1 <= strs.length <= 200</br>
0 <= strs[i].length <= 200</br>
strs[i]仅由小写英文字母组成</br>
