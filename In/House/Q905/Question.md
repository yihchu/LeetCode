<h1>按奇偶排序数组</h1>

给你一个整数数组nums，将nums中的的所有偶数元素移动到数组的前面，后跟所有奇数元素。
返回满足此条件的任一数组作为答案。

示例1：
输入：nums = [3, 1, 2, 4]
输出：[2, 4, 3, 1]
解释：[4, 2, 3, 1]、[2, 4, 1, 3]和[4, 2, 1, 3]也会被视作正确答案。

示例2：
输入：nums = [0]
输出：[0]

提示：
1 <= nums.length <= 5000
0 <= nums[i] <= 5000
