<h1>Pow(x, n)</h1>

实现pow(x, n)，即计算x的n次幂函数（即，x^n）。</br>

示例1：</br>
输入：x = 2.00000, n = 10</br>
输出：1024.00000</br>

示例2：</br>
输入：x = 2.10000, n = 3</br>
输出：9.26100</br>

示例3：</br>
输入：x = 2.00000, n = -2</br>
输出：0.25000</br>
解释：2^-2 = 1/2^2 = 1/4 = 0.25</br>

提示：
-100.0 < x < 100.0
-2^31 <= n <= 2^31-1
-10^4 <= x^n <= 10^4
