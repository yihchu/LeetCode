<h1>汉明距离</h1>

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。</br>
给出两个整数x和y，计算它们之间的汉明距离。</br>

注意：</br>
0 ≤ x, y < 2^31.</br>

示例:</br>
输入: x = 1, y = 4</br>
输出: 2</br>
解释:</br>

    1   (0 0 0 1)
    4   (0 1 0 0)
    ↑   ↑

上面的箭头指出了对应二进制位不同的位置。</br>
