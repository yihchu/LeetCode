<h1>字典序的第K小数字</h1>

给定整数n和k，返回[1, n]中字典序第k小的数字。</br>

示例1:</br>
输入: n = 13, k = 2</br>
输出: 10</br>
解释: 字典序的排列是[1, 10, 11, 12, 13, 2, 3, 4, 5, 6, 7, 8, 9]，所以第二小的数字是10。</br>

示例2:</br>
输入: n = 1, k = 1</br>
输出: 1</br>

提示:
- 1 <= k <= n <= 10^9
