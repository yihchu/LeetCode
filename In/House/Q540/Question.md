<h1>有序数组中的单一元素</h1>

给你一个仅由整数组成的有序数组，其中每个元素都会出现两次，唯有一个数只会出现一次。</br>
请你找出并返回只出现一次的那个数。</br>
你设计的解决方案必须满足O(logn)时间复杂度和O(1)空间复杂度。</br>

示例1:</br>
输入: nums = [1, 1, 2, 3, 3, 4, 4, 8, 8]</br>
输出: 2</br>

示例2:</br>
输入: nums =  [3, 3, 7, 7, 10, 11, 11]</br>
输出: 10</br>

提示:</br>
1 <= nums.length <= 10^5</br>
0 <= nums[i] <= 10^5</br>
