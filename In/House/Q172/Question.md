<h1>阶乘后的零</h1>

给定一个整数n，返回n!结果中尾随零的数量。</br>
提示n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1</br>

示例1：</br>
输入：n = 3</br>
输出：0</br>
解释：3! = 6，不含尾随0</br>

示例2：</br>
输入：n = 5</br>
输出：1</br>
解释：5! = 120，有一个尾随0</br>

示例3：</br>
输入：n = 0</br>
输出：0</br>

提示：
- 0 <= n <= 10^4
