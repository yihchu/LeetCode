<h1>最接近的三数之和</h1>

给你一个长度为n的整数数组nums和一个目标值target。请你从nums中选出三个整数，使它们的和与target最接近。</br>
返回这三个数的和。</br>
假定每组输入只存在恰好一个解。</br>

示例1：</br>
输入：nums = [-1, 2, 1, -4], target = 1</br>
输出：2</br>
解释：与target最接近的和是2(-1 + 2 + 1 = 2)。</br>

示例2：</br>
输入：nums = [0, 0, 0], target = 1</br>
输出：0</br>

提示：</br>
3 <= nums.length <= 1000</br>
-1000 <= nums[i] <= 1000</br>
-10^4 <= target <= 10^4</br>
