<h1>最小绝对差</h1>

给你个整数数组arr，其中每个元素都不相同。</br>
请你找到所有具有最小绝对差的元素对，并且按升序的顺序返回。</br>

示例1：</br>
输入：arr = [4, 2, 1, 3]</br>
输出：[[1, 2], [2, 3], [3, 4]]</br>

示例2：</br>
输入：arr = [1, 3, 6, 10, 15]</br>
输出：[[1, 3]]</br>

示例3：</br>
输入：arr = [3, 8, -10, 23, 19, -4, -14, 27]</br>
输出：[[-14, -10], [19, 23], [23, 27]]</br>

提示：
- 2 <= arr.length <= 10^5
- -10^6 <= arr[i] <= 10^6
