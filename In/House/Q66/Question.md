<h1>加一</h1>

给定一个由整数组成的非空数组所表示的非负整数，在该数的基础上加一。</br>
最高位数字存放在数组的首位，数组中每个元素只存储单个数字。</br>
你可以假设除了整数0之外，这个整数不会以零开头。</br>

示例1：</br>
输入：digits = [1, 2, 3]</br>
输出：[1, 2, 4]</br>
解释：输入数组表示数字123。</br>

示例2：</br>
输入：digits = [4, 3, 2, 1]</br>
输出：[4, 3, 2, 2]</br>
解释：输入数组表示数字4321。</br>

示例3：</br>
输入：digits = [0]</br>
输出：[1]</br>

提示：</br>
1 <= digits.length <= 100</br>
0 <= digits[i] <= 9</br>
