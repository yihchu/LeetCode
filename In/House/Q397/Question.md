<h1>整数替换</h1>

给定一个正整数n ，你可以做如下操作：</br>
如果n是偶数，则用n / 2替换n。</br>
如果n是奇数，则可以用n + 1或n - 1替换n。</br>
n变为1所需的最小替换次数是多少？</br>

示例1：</br>
输入：n = 8</br>
输出：3</br>
解释：8 -> 4 -> 2 -> 1</br>

示例2：</br>
输入：n = 7</br>
输出：4</br>
解释：7 -> 8 -> 4 -> 2 -> 1</br>
或 7 -> 6 -> 3 -> 2 -> 1</br>

示例3：</br>
输入：n = 4</br>
输出：2</br>

提示：</br>
1 <= n <= 2^31 - 1</br>
