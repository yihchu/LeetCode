<h1>合并两个有序链表</h1>

将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。<br>

示例1：<br>
</br>![](./image/1.jpg)</br></br>
输入：l1 = [1, 2, 4], l2 = [1, 3, 4]<br>
输出：[1, 1, 2, 3, 4, 4]<br>

示例2：<br>
输入：l1 = [], l2 = []<br>
输出：[]<br>

示例3：<br>
输入：l1 = [], l2 = [0]<br>
输出：[0]<br>

提示：
- 两个链表的节点数目范围是[0, 50]
- -100 <= Node.val <= 100
- l1和l2均按非递减顺序排列
