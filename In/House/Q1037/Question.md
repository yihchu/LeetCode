<h1>有效的回旋镖</h1>

给定一个数组points，其中points[i] = [x[i], y[i]]表示X-Y平面上的一个点，如果这些点构成一个回旋镖则返回true。</br>
回旋镖定义为一组三个点，这些点各不相同且不在一条直线上。</br>

示例1：</br>
输入：points = [[1, 1], [2, 3], [3, 2]]</br>
输出：true</br>

示例2：</br>
输入：points = [[1, 1], [2, 2], [3, 3]]</br>
输出：false</br>

提示：</br>
points.length == 3</br>
points[i].length == 2</br>
0 <= x[i], y[i] <= 100</br>
