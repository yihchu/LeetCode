<h1>最少移动次数使数组元素相等II</h1>

给你一个长度为n的整数数组nums，返回使所有数组元素相等需要的最少移动数。
在一步操作中，你可以使数组中的一个元素加1或者减1。

示例1：
输入：nums = [1, 2, 3]
输出：2
解释：
只需要两步操作（每步操作指南使一个元素加1或减1）：
[1, 2, 3] => [2, 2, 3] => [2, 2, 2]

示例2：
输入：nums = [1, 10, 2, 9]
输出：16

提示：
n == nums.length
1 <= nums.length <= 10^5
-10^9 <= nums[i] <= 10^9
